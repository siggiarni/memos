# MacOS keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `⌃a` | Go to start of a line |
| `⌃e` | Go to end of line |
| `⌃p` | Go up one line |
| `⌃n` | Go down one line |
| `⌃k` | Cut line proceeding cursor |
| `⌃f` | Forward one char |
| `⌃b` | Back one char |
| `⌃f` | Forward one char |
| `⌃t` | Swap the left and right characters adjacent to cursor |
